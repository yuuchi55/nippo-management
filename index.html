日報管理システム

中小企業向けの標準的な日報管理ツールです。

⚠️ 注意

このシステムはNode.js/Express.jsサーバーが必要なため、GitHub Pagesでは動作しません。

ローカル環境でのセットアップが必要です。

🚀 セットアップ方法

git clone https://github.com/yuuchi55/nippo-management.git
cd nippo-management
npm install
npm start

その後、http://localhost:3001 でアクセスできます。

✨ 機能

  • 📝 日報の作成・編集・削除
  • 🔍 日付、社員ID、部署での検索・フィルター
  • 📊 CSVエクスポート機能
  • 📱 レスポンシブデザイン

🛠️ 技術スタック

  • バックエンド: Express.js + SQLite
  • フロントエンド: HTML/CSS/JavaScript
  • データベース: SQLite（自動作成）
